HTML

HTML stands for "Hypertext Markup Language". Its what I used to create this website!

Using HTML, you can add lists, tables, pictures and videos. You can also style HTML using CSS

To create a basic structure of HTML:

  • Start the browser document with the !DOCTYPE tag
  • After that, the html tag must be added
  • Then you can declare a head section and a body section
  • Close out the HTMLt tag
